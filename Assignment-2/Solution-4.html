List of some of the many HTML Tags:

  1. <html> Tag: The <html> tag represents the root of an HTML document. The <html> tag is the container of all the HTML elements (except for <!DOCTYPE> Declaration)

  2. <head> Tag: The <head> element is a container for metadata (data about data) and is placed between <html> tag and <body> tag.

  3. <meta> Tag: The <meta> tag defines the metadata about an HTML document. Metadata is data about data. <meta> tags are always placed inside the <head> element , and are used to specify character set, page description, keywords, viewport settings, and author of the document.

  4. <body> tag: The <body> tag defines the document's body. The <body> element contains all the contents of an HTML document, such as heading, paragraphs, images, hyperlinks, tables, lists, etc.

  5. <h1> to <h6> Tags: The <h1> to <h6> tags are used to define HTML headings. <h1> tag defines the most important heading. <h6> tag defines the least important heading.

  6. <p> Tag: The <p> tag defines a paragraph. Browsers automatically add a blank single line before anf after each <p> element.

  7. <br> Tag: The <br> tag inserts a single line break. It is useful for writing addresses or poems. The <br> tag is an empty tag which means it has no end tag.

  8. <div> Tag: The <div> tag defines a division or a section in an HTML document. The <div> tag is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript

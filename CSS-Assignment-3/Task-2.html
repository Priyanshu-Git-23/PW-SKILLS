Difference between Absolute and Relative Positioning
Absolute Positioning Relative Positioning
An element with position: absolute; is positioned relative to the nearest positioned ancestor. However, if there's no positioned ancestor, it uses the document body. An element with postion: relative; is positioned relative to its normal position.
Absolute positioned elements are removed from the normal flow, and can overlap elements. Other content will not be adjusted to fit into any gap left by an element.

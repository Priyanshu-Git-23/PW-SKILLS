Features of HTML5 are:
     1. Intro of audio and video tags- Audio and video tags are two major additions in HTML5. It allows developers to embed audio and video on their website. With the help of CSS, we can change the border, opacity, gradient, reflections, transitions, transformations and even animations. HMTL5 makes adding video super fast.
     2. Header and footer- With these new tags, there is no need to identify the two elements with a <div> tag. Footer is placed at the end of the web page while the header is placed the start of the web page. By using <header> and <footer> HTML5 elements, the browser will know what to load first and what to load later.
     3. Figure and figcaption- HTML5 allows to use a <figure> element to mark up a photo in a document, and a <figcaption> element to define a caption for a photo. The <figcaption> tag defines a caption for a <figure> element.
     4. Nav tag- The <nav> tag defines a set of navigation links. It is used for the part of internet site that links to different pages at the website. COmmon examples of nav elements are menus, tables, contents, and indexes. Earlier, the <div> tag was used with the id or class attribute with a value nav.
     5. Placeholder attribute- The placeholder attribute specifies a short hint that defines the expected value of an input field/text area. The short hint is displayed in the field before the user enters the value.
